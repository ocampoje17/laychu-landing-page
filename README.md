## Cong nghe

- React 19
- TypeScript
- Vite
- HeroUI
- Tailwind CSS 4
- `@heroicons/react`

## Chay local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run lint
```

## GitHub Pages

- Repository Pages URL hien tai: `https://ocampoje17.github.io/laychu-landing-page/`

- Workflow deploy tu dong nam tai `.github/workflows/deploy.yml`
- Trong GitHub: vao `Settings` -> `Pages` -> chon `Source` la `GitHub Actions`
